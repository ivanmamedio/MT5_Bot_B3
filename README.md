MT5_Bot_B3
Robô de Day Trade para B3 (Mini Índice - WIN e Mini Dólar - WDO) desenvolvido para o MetaTrader 5.

Estrutura do Projeto
Experts: Robôs (EAs) principais e auxiliares.

Indicators: Indicadores customizados.

Include: Classes, structs, enums e bibliotecas (OOP).

Core: Lógica de execução, gerenciamento de ordens.

Signals: Classes de geração de sinais (Estratégias).

Risk: Gestão de risco e cálculo de lotes.

Utils: Funções auxiliares, logs, formatação.

Scripts: Scripts para execução única.

Python: Scripts Python para análise, ML ou automação externa.

Tests: Backtests, relatórios e dados de validação.

Docs: Documentação, regras de negócio, diário de trading.

Presets: Arquivos .set de otimização do Strategy Tester.

Hardware Target
CPU: Intel Core i7-13620H (10 cores)

RAM: 16GB DDR5

Storage: SSD NVMe M.2
